# 💻 Installation
**Step 1:** `git clone https://github.com/dsk90it/recipe-app.git`

**Step 2:** Switch to cloned folder and run `npm install` in your terminal

**Step 3:** `npm run dev-server` to start server

**For Prod Build :** `npm run build`
